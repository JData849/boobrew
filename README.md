# Boo&Brew Next.js Homepage Starter

A dependency-light Next.js App Router homepage starter for Boo&Brew: personalised tumblers, insulated tumblers, cold cups, mugs, key rings, gift sets, seasonal designs and custom orders.

## Run

```bash
npm install
npm run dev
```

## Included

- `app/page.tsx`
- `app/layout.tsx`
- `app/globals.css`
- Typed homepage data
- Semantic homepage sections
- Header, footer, search drawer and cart drawer
- Product cards, carousel, gallery, reviews, newsletter and made-to-order sections
- CSS tokens, responsive layout, accessibility basics and reduced-motion support

## Images

Add real images to:

```txt
/public/images/home/
/public/images/products/
```

Or update the image paths in:

```txt
components/homepage/homepage.data.ts
```
